# Build instructions

1. Install [nuke](https://nuke.build/docs/getting-started/setup.html)
2. Run `nuke`

# Preview documentation

1. Install [docfx](https://dotnet.github.io/docfx/tutorial/docfx_getting_started.html)
2. Run `docfx ./docfx_project/docfx.json --serve`
